# **App Name**: Carta Ya!

## Core Features:

- Landing Page: A public landing page showcasing the benefits of the platform and offering a clear call to action to create a store or start a free trial.
- Store Registration and Management: A registration system for store owners to create and manage their store profiles, including essential information such as contact details, address, and store name.
- Category Management: Functionality for store owners to create, edit, and delete product categories, each with a name and image, to organize their online menu effectively.
- Product Management: A robust product management system allowing store owners to add, modify, and remove products, including details such as image, name, price, and optional variants with price adjustments.
- Order Placement via WhatsApp: Enables customers to place orders directly through WhatsApp without needing to register, automatically generating a pre-formatted message with order details.
- Subscription Management: A subscription system that grants full access upon registration with a 7-day free trial, after which access is limited until a subscription payment is confirmed via an external link.
- AI-Powered Product Description Tool: An AI-powered tool that assists vendors with writing effective product descriptions based on basic product information and attributes.

## Style Guidelines:

- Color scheme: A dark theme is used as the default. Background is a very dark gray (#121212). Primary color is a vibrant yellow (#FFDA63), symbolizing energy and modernity.
- Accent color: Use a complementary orange (#FFB347) to create contrast. Useful for calls to action or key interactive elements.
- Font choice: 'Poppins' (sans-serif) is the primary font for both headings and body text due to its clean and modern feel. Note: currently only Google Fonts are supported.
- Mobile-first responsive design ensuring a seamless experience on various devices. Intuitive and straightforward navigation is crucial, tailored for non-technical users.
- Simple and clear icons to represent categories and actions. Designed to be easily understandable at a glance.
